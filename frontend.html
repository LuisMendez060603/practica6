Frontend


La definición de "Front-end" en el contexto de desarrollo web se refiere a la parte de un sitio web o aplicación web que los usuarios finales ven y con la que interactúan directamente en su navegador web. El término "front-end" también se conoce como "cliente" o "capa de presentación". El desarrollo front-end se centra en la creación y mejora de la interfaz de usuario, la experiencia del usuario (UX) y la apariencia visual de un sitio web o aplicación web. Esto implica utilizar tecnologías como HTML, CSS y JavaScript para estructurar y diseñar el contenido web, así como para habilitar la interactividad y la funcionalidad del lado del cliente.


Los fundamentos del desarrollo front-end implican comprender los principios y las tecnologías básicas que permiten crear interfaces de usuario web efectivas y atractivas. Aquí están los fundamentos clave:

  • HTML (Lenguaje de marcado de hipertexto): HTML es el lenguaje fundamental para crear la estructura y el contenido de una página web. Los conceptos clave incluyen etiquetas HTML (como html, head, body, div, p, h1, a, etc.), la clasificación de elementos, la semántica web (uso adecuado de etiquetas para describir el contenido) y la accesibilidad.
  • CSS (Hojas de estilo en cascada): CSS se utiliza para dar estilo y formato al contenido HTML.
